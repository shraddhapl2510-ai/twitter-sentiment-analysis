# 🐦 Twitter Sentiment Analysis

Ever wondered if a tweet is happy or sad? This project uses Machine Learning to find out! 

This notebook analyzes Twitter data to classify sentiments as **Positive** or **Negative**. It's a great starter project for Natural Language Processing (NLP) and Scikit-Learn.

## ⚡ Quick Stats
- **Model:** Logistic Regression
- **Vectorization:** TF-IDF (5000 features)
- **Accuracy:** ~79%
- **Language:** Python
